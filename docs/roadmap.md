# BidWave Roadmap

## Phase 1: Foundation

- finalize product scope
- define legal and risk requirements
- set up monorepo, shared types, and database schema
- select providers for video, auth, and payments

## Phase 2: Marketplace core

- build auth and user profiles
- build seller application flow
- create show scheduling and show detail pages
- build live room shell with stream slot and chat

## Phase 3: Transaction engine

- build auction lifecycle
- build realtime bids over websockets
- add payment method collection
- create checkout and order creation

## Phase 4: Operations

- seller dashboard
- fulfillment workflow
- payout scheduling
- moderation queue and support views

## Phase 5: Pilot launch

- onboard first sellers
- shadow first shows manually
- monitor latency and payment failures
- refine fraud and support policies
